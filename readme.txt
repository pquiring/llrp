LLRP (l4kjava) fork "as is".

Building:
  - check out javaforce
    git clone http://github.com/pquiring/javaforce
  - create lib folder
    cd javaforce
    md lib
    cd lib
  - check out llrp
    git clone http://github.com/pquiring/llrp
  - build it
    cd llrp
    ant

Notes:
  - JavaForce build scripts are only used to download dependancies
